# Movie Recommendation System

This project is a content-based movie recommendation system built using machine learning.

## Features
- Recommends top 5 similar movies
- Uses cosine similarity
- Based on TMDB dataset

## Tech Stack
- Python
- Pandas
- Scikit-learn

## How it works
- Combines movie features (genres, keywords, overview)
- Converts text into vectors using CountVectorizer
- Computes cosine similarity between movies

## Run
python recommender.py
